# Cisco presets

This preset directory contains Cisco data sources ([Umbrella logs](https://docs.umbrella.com/)). Currently supported data sources:

- [umbrella_dns](https://docs.umbrella.com/umbrella-user-guide/docs/dns-log-formats)

## Umbrella DNS

Assumed data format: csv (no header)

Silver tables:
- cisco_umbrella_dns

Gold tables:
- dns_activity
